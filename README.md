# CEE6501-Project
This is the final project for CEE 6501 Matrix Structural Analysis which will implement the Direct Stiffness Method (DSM) into Python.

## Features
- 3D Truss Elements
- 3D Frame elements
- Mixed element types
- JSON input file
- Ouputs containing displacements, reactions, and forces

## How to run
1. Open 'main.ipynb'
2. Modify input file in inputs folder
3. Runn all cells

## Structure
- The helpers folder will contain the main functions
- The inputs folder will contain the models I will be using
- The outputs folder will store the results
